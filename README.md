模板缓冲:
	1、启用模板缓冲的写入。
	2、渲染物体，更新模板缓冲的内容。
	3、禁用模板缓冲的写入。
	4、渲染（其它）物体，这次根据模板缓冲的内容丢弃特定的片段。

骨骼动画:
	1、骨骼空间与模型空间
	aiBone->mOffsetMatrix : 从模型空间变换到骨骼空间的变换矩阵
	aiNode->mTransformation : 从自己的骨骼空间变换到父亲节点的骨骼空间

	Model Space * mOffsetMatrix ---> Bone Space * mTransformation ---> Parent Bone Space --->...............
